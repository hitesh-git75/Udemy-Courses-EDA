# Udemy-Courses-EDA

## Project Overview
This project explores the Udemy Courses dataset using Python to uncover insights about course popularity, ratings, pricing, and subscriber engagement.

## Objectives
- Analyze course performance and trends
- Identify top-selling and highest-rated courses
- Explore relationships between price, subscribers, and ratings
- Generate meaningful visualizations

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Findings
- Identified the most popular courses based on subscribers.
- Analyzed course pricing patterns.
- Explored factors influencing course ratings and engagement.
- Created visualizations to present insights effectively.

## Dataset
Udemy Courses Dataset
